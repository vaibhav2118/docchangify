FAQ

Is my PDF safe?

Yes! All processing happens in your browser using PDF.js. No file is uploaded to any server.

Does it work with scanned PDFs?

This tool extracts digital text layers. Scanned/image-based PDFs may return little or no text — OCR is not supported.

What is the file size limit?

There is no hard limit — it depends on your browser memory. Very large PDFs (100+ pages) may take a moment.
